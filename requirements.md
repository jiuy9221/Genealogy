# Genealogy Web App - Requirements (v1)
离线族谱应用（纯前端，无后端）

## 核心需求
1. **人员管理**
   - 添加人员（姓名、性别、出生、死亡、备注）
   - 编辑人员
   - 删除人员
   - 列表显示所有人员

2. **关系管理**
   - 添加父母
   - 添加子女
   - 添加配偶
   - 自动构建家庭结构

3. **族谱树展示**
   - 使用 D3.js 或 Canvas 绘制树结构
   - 支持无限代
   - 父母在上，子女在下
   - 配偶横向连接

4. **数据存储（双格式）**
   - family.json（结构化数据）
   - family.md（Markdown 轻量文本）
   - 数据双向转换（md <-> json）
   - 自动本地缓存（localStorage）

5. **数据操作功能**
   - 导入 JSON
   - 导入 Markdown
   - 导出 JSON
   - 导出 Markdown
   - 自动保存按钮

6. **UI 需求**
   - 左侧：人员列表
   - 中间：族谱树显示
   - 右侧：人员信息编辑
   - 顶部按钮：导入 / 导出 / 新增人员

7. **离线运行要求**
   - index.html 在没有网络情况下可直接使用
   - 所有 JS、CSS、本地图标必须本地引用

## 后续增强（future）
- 多族谱文件管理
- 多语言切换（繁体中文 / English）
- 祖先路径显示（选中某人后显示到根节点的路径）
- 触摸设备双指缩放（pinch-zoom）
